# 📖 Introduction

Security Operations Centers (SOCs) play a critical role in protecting organizations from cyber threats. SOC Analysts monitor, detect, investigate, and respond to security incidents using various cybersecurity tools and methodologies.

This repository provides an overview of major cybersecurity domains relevant to SOC analysts along with commonly used tools and technologies.

---

# 👨‍💻 SOC Analyst Responsibilities

A SOC Analyst is responsible for:

- Monitoring security alerts and events
- Investigating suspicious activities
- Performing incident triage
- Responding to security incidents
- Conducting threat hunting
- Managing SIEM dashboards
- Analyzing malware and phishing attempts
- Coordinating with security teams

---

# 🔐 Core Cybersecurity Domains

---

## 🎣 Phishing Analysis

Phishing analysis involves identifying and investigating malicious emails, fake websites, and social engineering attacks designed to steal sensitive information.

### Key Areas:
- Email header analysis
- URL inspection
- Attachment sandboxing
- Domain reputation checks
- IOC extraction

### Common Tools:
- VirusTotal
- Any.Run
- PhishTool
- URLScan.io
- MXToolbox

---

## 🌐 Network Security Monitoring

Network security focuses on monitoring and protecting network traffic from unauthorized access, malware, and attacks.

### Key Areas:
- Packet analysis
- Intrusion detection
- Firewall monitoring
- Traffic anomaly detection

### Common Tools:
- Wireshark
- Zeek
- Suricata
- Snort
- tcpdump

---

## 💻 Endpoint Security

Endpoint security protects devices such as laptops, servers, and workstations from cyber threats.

### Key Areas:
- Endpoint Detection & Response (EDR)
- Antivirus monitoring
- Process analysis
- Behavioral detection

### Common Tools:
- CrowdStrike Falcon
- Microsoft Defender
- SentinelOne
- Carbon Black

---

## 📊 SIEM & Log Analysis

Security Information and Event Management (SIEM) systems collect and analyze logs from various sources to detect threats.

### Key Areas:
- Log correlation
- Alert triage
- Dashboard monitoring
- Rule creation

### Common Tools:
- Splunk
- IBM QRadar
- Microsoft Sentinel
- Elastic Stack
- ArcSight
